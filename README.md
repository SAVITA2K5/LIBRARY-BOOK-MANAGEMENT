# Library Book Management System

A full-stack Library Book Management System built using **MongoDB, Express.js, Node.js, and React**.  
The project demonstrates **CRUD operations**, **schema validation**, **REST APIs**, and **frontend–backend integration**.

## Features
- Add, view, update, and delete books  
- Filter books by category  
- View books published after a specific year  
- Prevent negative stock updates  
- Conditional deletion when available copies = 0  

## Tech Stack
- **Backend:** Node.js, Express.js, MongoDB, Mongoose  
- **Frontend:** React (Vite), Axios  
- **Tools:** Postman, Git, GitHub  

## How to Run
### Backend
```bash
cd backend
node server.js
# Frontend
cd frontend
npm run dev
